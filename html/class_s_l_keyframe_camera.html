Detailed Description

Special camera for ORB-SLAM keyframes that allows the video image display on the near clippling plane.

SLKeyframeCamera::drawMeshes draws the cameras frustum lines.

Only draws the frustum lines without lighting when the camera is not the active one. This means that it can be seen from the active view point.

Reimplemented from SLCamera.
