Detailed Description

Photonmap class with datastructure, scattering and gattering methods.

Holds a datastructure for storing photons and diffrent methods for the photon scattering and the radiance estimate. The datastructure is a heaplike left- balanced kd-tree. It is implented following the source code from H. W. Jensen described in his book "Realistic Image Synthesis Using Photon Mappin.
