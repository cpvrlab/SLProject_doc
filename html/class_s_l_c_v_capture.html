Detailed Description

Encapsulation of the OpenCV Capture Device.

It holds a static image for the last captured color frame and a grayscale version as well as a single static instance of the OpenCV capture device. The The live video image grabbing is not mandatory and can be replaced by the the top level application with its own video grabbing functionality. This is e.g. used in the iOS or Android examples. The SLCVCapture::lastFrame and SLCVCapture::lastFrameGray are on the other hand used in all applications as the buffer for the last captured image.
