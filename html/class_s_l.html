void SL::parseCmdLineArgs ( SLVstring &  cmdLineArgs)
static

Parses the command line arguments and sets the according scene test variable.

The following command line arguments can be passed:

  • testScene: scene int ID defined in the enum SLCommand (0=sceneAll)
  • testDurationSec: test duration in int sec.
  • testFactor: test int factor used for scaling in SLScene::onLoad.

    Example:\ app_Demo.exe testScene=1 testDurationSec=5 testFactor=1

    Starts the app with scene 1 (=sceneMinimal) and runs for 5 sec. with testFactor 1. If the testScene is 0 (=sceneAll) all scenes are tested one after the other.
    The scenes are changed and logged in SLSceneView::testRunIsFinished() that is called in SLSceneView::onPaint().
Parameters
cmdLineArgsA string vector with all command line arguments.
