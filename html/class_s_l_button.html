Detailed Description

Defines a 2D-GUI button with optional submenus as children of the group.

SLButton defines a GUI button with optional submenus as children of the group. It serves as a minimal hierarchical GUI element for implementing a menu. It has a SLText instance for text rendering. The button can be either a click-, check- or radio-button. For check and radio button there is a cross drawn in checked state. A button can have a command (SLCommand) associated. A button without command is expected to open its children button as a submenu. The button tree is created in SLSceneView::build2DMenus and attached to SLScene._menu2D. Opening, closing & command execution is handled in onMouseDown and onMouseUp.

If closeOnClick is true the entire menu gets closed after command execution.
