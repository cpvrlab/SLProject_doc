Detailed Description

template<class T>
class SLAverage< T >

SLAverage template class provides an average value from a fixed size array.

The SLAverage template class provides an average value continuously averaged from a fixed size array. It is used as SLAvgFloat for averaging time keeping in SLSceneView.
