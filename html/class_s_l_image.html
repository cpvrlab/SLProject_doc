Detailed Description

Small image class for loading JPG, PNG, BMP, TGA and saving PNG files.

Minimal class for loading JPG, PNG, BMP, TGA and saving PNG files. In addition you can fill, resize, flip and convolve an image. The class is used in SLGLTexture. SLImage interpretes an image starting at bottom left as OpenGL.
