Detailed Description

Texture object generated in run time from another texture.

This class is mainly used to generate the textures used for reflections in the IBL techniques. It takes a source texture and projects it into a cube map with different rendering techniques to produces the textures needed in IBL like the irradiance or the prefilter map. This textures can be given to the SLMaterial with IBL. It uses the SLGLFrameBuffer class to render the scene into a cube map.
