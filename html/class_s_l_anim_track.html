Detailed Description

Abstract base class for SLAnimationTracks providing time and keyframe functions.

An animation track is a specialized track that affects a single SLNode or an SLJoint of an SLSkeleton by interpolating its transform. It holds therefore a list of SLKeyframe. For a smooth motion it can interpolate the transform at a given time between two neighboring SLKeyframe.
